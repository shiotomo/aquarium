# aquarium

## このリポジトリについて

Dockerコンテナ上でデータベースを管理するためのツールです。

## 使用技術

- Docker
- DockerCompose

## 使用データベース

- PostgreSQL 9.5
- PostgreSQL 12.2
- MySQL 5.7
- MySQL 8.0
- Redis 3.2.9
- Redis 5.0.5

## コマンド

実行
```
docker-compose up
```

停止
```
docker-compose down
```
